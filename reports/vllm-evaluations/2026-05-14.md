## 미해결 이슈

1. **GPU TTFT 실측 미수행**: CPU 환경이므로 TTFT p50 +10% 이내 기준을 정량 실측하지 못했다. `encode_count`/`decode_count` 추적은 구현되었으나 wall-clock latency는 GPU 환경에서 별도 측정 필요.

2. **SwigPy DeprecationWarning**: `SwigPyPacked`/`SwigPyObject` 관련 `DeprecationWarning` 2건이 vllm 패키지 내부 SWIG C extension에서 발생한다. 통합 코드 자체에서 발생하는 경고가 아니므로 통합 코드 범위 내 수정 불가.

3. **Report ① Activity C config 의존성**: 10x config(bits_dir=4, 3.56x actual)에서 attention error 13.1%로 mandatory 임계값 초과. 이는 Spec.md에 명시된 설계 결정이며, 10x tier에서는 cosine >= 0.97 proxy가 적용된다.

4. **4 skipped 테스트**: `TestNonContiguousKVCacheManagerV2`는 full vLLM infra(BlockPool, KVCacheConfig) 없이는 실행 불가하여 스킵됨. 이 패턴은 모든 이전 사이클과 일관된다.
