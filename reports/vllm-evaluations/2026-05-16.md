## 미해결 이슈

**Partial 항목 (필수 아님, 다음 사이클 참고용):**

1. **실제 GPU Throughput 측정 불가**: GPU가 없는 CI 환경에서 tokens/sec 실측 불가. 스케줄링 오버헤드(p50=0.10ms)와 압축 오버헤드(p50=2.21ms)로 간접 검증. 실제 처리량 측정은 GPU 환경 E2E 실험에서 필요.

2. **Tier 3 INT8 per-element relative error**: 평균 절댓값/평균 크기 기준 0.94% (pass), element-wise 기준 3.6% (분모 근사 zero에 의한 수치 불안정성). install.sh 스모크 테스트 기준 PASS. 정확도 보존 기준 ±1%는 attention output proxy로 별도 검증되어 있어 실제 정확도 위협 없음.

3. **Wikipedia/LongBench 실제 perplexity 측정 미완**: 실제 언어 모델 없이 attention output proxy 검증. 실제 perplexity 측정을 위해 Llama-3-8B 등 소형 모델 + GPU 환경 필요.

4. **멀티-GPU tensor parallel 검증**: 단일 프로세스 환경에서 멀티-GPU 병렬 동작 확인 불가. vLLM tensor_parallel_size=2 설정의 블록 매핑 일관성은 GPU 환경에서만 검증 가능.

**필수 항목 Fail**: 없음.
