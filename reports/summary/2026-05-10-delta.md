## 주요 신규 컴포넌트

### B: KVPacketSoftAdapterCache (`src/cache/kv_packet_adapter.py`)
- SoftTokenAdapter 2-layer MLP로 재계산 없는 비연속 KV 재사용
- SHA-256 위치-독립 키로 임의 세그먼트 독립 조회
- pack_packets() 경로: cat+adapt만 수행, 오버헤드 <<5% 설계 보장
- LRU eviction max_packets=512 캡

### C: VQCodec (`src/compression/vq_codec.py`)
- training-free k-means 벡터 양자화, pre-RoPE 코드북
- codebook_size=64, n_residuals=4, recent_window=8 FP16 보존
- inverse_rope/apply_rope 정적 메서드 분리 설계
- 첫 write 시 auto-fit, 재현성 seed=42 보장

### B+C 통합: ContextFreeCompressedKVPacket (`src/cache/context_free_compressed_packet.py`)
- CompressedPacket dataclass로 저장 구조 명시
- put_compressed / get_decompressed 분리 인터페이스
- CacheStore 6개 추상 메서드 완전 구현

### A(보조): KVPacketSegmentSchedulerMixin (vLLM 이식)
- pre_schedule_kvp() 세그먼트 인덱스 기반 요청 우선순위 정렬
- overhead_budget_ms 초과 시 즉시 break, 오버헤드 상한 보장
- make_kvp_segment_scheduler_class() 팩토리 API
